# Sample---ISMS
ISMS / Security GRC Portfolio

This repository contains a conceptual example of an Information Security Management System (ISMS) based on ISO 27001:2022.  
It is designed as a demonstration project to showcase skills in governance, risk management, compliance, and documentation.

## Contents
- Scope definition
- Asset register
- Risk assessment methodology
- Example risk register
- Statement of Applicability (SoA)
- Sample policies
- Audit checklist
- Templates for governance and reporting

This project contains no real company data and is intended purely for demonstration and educational purposes.
